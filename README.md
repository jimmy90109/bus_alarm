# 畢業專題 - 公車定位鬧鐘APP  

藉由到站下車APP，實時的手機定位資訊，不會再因為塞車而需要再設另一個鬧鐘、到站時的連續震動除非使用者關閉，否則會繼續震動，確保使用者已經確實從睡夢中清醒過來，讓通勤族不會再睡得提心吊膽，大大提升在交通工具上的睡眠品質。
